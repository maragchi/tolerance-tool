# FLOW-03: 智能配合推荐

## 目标
用户选择工况条件，工具推荐最适合的配合代号，并解释原因。

## 前置条件
- FLOW-01 完成
- FLOW-02 完成（基础查询可用）

## 功能设计

### 输入（用户通过选项选择）
1. **配合类型**：间隙配合 / 过渡配合 / 过盈配合 / 不确定
2. **运动条件**：有相对运动 / 无相对运动
3. **载荷条件**：轻载 / 中载 / 重载 / 无载荷
4. **拆卸需求**：需要经常拆卸 / 偶尔拆卸 / 不拆卸
5. **定心精度**：高精度 / 一般精度 / 低精度

### 输出
1. 推荐的配合代号（优先推荐1个，备选2-3个）
2. 每种推荐的理由
3. 装配方法提示
4. 典型应用场景
5. "点击查看详细公差值" → 跳转 FLOW-02 的查询结果

## 实现步骤

### Step 1: 创建 fit-recommender.js
决策逻辑（三层）：

**第一层：工况 → 配合大类**
```
有相对运动 → 间隙配合
传递扭矩 + 不拆卸 → 过盈配合
精确定位 + 可拆卸 → 过渡配合
```

**第二层：工况细节 → 具体配合代号**
```
间隙配合 + 高精度 → H7/g6
间隙配合 + 一般精度 → H8/f7
间隙配合 + 粗糙 → H9/d9 / H11/c11
过渡配合 + 经常拆卸 → H7/js6 / H7/h6
过渡配合 + 偶尔拆卸 → H7/k6
过渡配合 + 不拆卸 → H7/m6 / H7/n6
过盈配合 + 中等载荷 → H7/p6 / H7/r6
过盈配合 + 重载荷 → H7/s6
```

**第三层：显示推荐详情**
从 fit-recommendations.json 查询该配合的完整信息。

### Step 2: 创建推荐 UI
- 工况选择区：用下拉框或按钮组
- 推荐结果区：卡片式展示
- 每张卡片包含：配合代号、配合特性、装配方法、应用举例

### Step 3: 集成基础查询
- 推荐结果中添加"查看详细公差"按钮
- 点击后弹出基本尺寸输入框
- 计算并显示该配合的极限偏差和间隙/过盈

## 测试用例

| 工况 | 期望推荐 |
|------|---------|
| 滑动轴承，有相对运动，高精度 | H7/g6 |

| 法兰定位，无载荷，需拆卸 | H7/h6 |
| 火车轮箍，重载冲击 | H7/s6 |
