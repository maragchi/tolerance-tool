# FLOW-02: 基础公差查询

## 目标
用户输入基本尺寸 + 孔/轴公差代号，工具计算出极限偏差和极限尺寸。

## 前置条件
- FLOW-01 完成（JSON数据文件就位）

## 功能设计

### 输入
1. 基本尺寸（mm），如 50
2. 公差代号，如 H7（孔）或 g6（轴）

### 输出
1. 上偏差 ES/es
2. 下偏差 EI/ei
3. 最大极限尺寸
4. 最小极限尺寸
5. 公差值

### 示例
输入：基本尺寸 50mm，公差代号 H7
输出：
- 上偏差 ES = +0.025mm
- 下偏差 EI = 0mm
- 最大极限尺寸 = 50.025mm
- 最小极限尺寸 = 50.000mm
- 公差 = 0.025mm

## 实现步骤

### Step 1: 创建 index.html
- 基本页面结构：输入区 + 结果区
- 两个输入框：基本尺寸、公差代号
- 一个"查询"按钮
- 结果展示区

### Step 2: 创建 tolerance-calculator.js
- `getToleranceGradeValue(dim, itGrade)` — 根据尺寸分段查IT公差值
- `getFundamentalDeviation(dim, letter)` — 根据尺寸分段查基本偏差
- `calculateTolerance(dim, toleranceCode)` — 组合计算极限偏差
  - 解析公差代号（字母 + IT等级数字）
  - 判断孔（大写）还是轴（小写）
  - 查表得出基本偏差和IT值
  - 计算另一偏差

### Step 3: 连接UI和计算逻辑
- 输入验证（尺寸范围 1-3150mm，代号格式正确）
- 查询按钮点击 → 调用计算 → 显示结果

### Step 4: 配合查询（进阶）
- 支持输入配合代号如 "H7/g6"
- 同时计算孔和轴的偏差
- 计算最大间隙/过盈、最小间隙/过盈
- 判断配合类型（间隙/过渡/过盈）

## 测试用例

| 输入 | 期望上偏差 | 期望下偏差 | 期望公差 |
|------|-----------|-----------|---------|
| 50 H7 | +0.025 | 0 | 0.025 |
| 50 g6 | -0.009 | -0.025 | 0.016 |
| 50 H7/g6 (配合) | Max间隙 +0.050 | Min间隙 +0.009 | 间隙配合 |

验证方式：与纸质公差表或 mechtool.cn 在线查询结果交叉对比。
