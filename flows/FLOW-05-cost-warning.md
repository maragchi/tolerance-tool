# FLOW-05: 后果提示（成本+加工）

## 目标
用户选择公差等级后，提示对应的加工方法、表面粗糙度、成本增幅和常见后果。

## 前置条件
- FLOW-01 完成

## 功能设计

### 触发方式
1. 在基础查询结果中自动显示（选择公差等级后）
2. 在配合推荐结果中自动显示
3. 也可独立使用：选择IT等级 → 查看详情

### 输出内容
1. **可达加工方法** — 如 IT7 = 精车、精铣、粗磨、铰孔
2. **表面粗糙度 Ra** — 如 IT7 = 0.8~1.6 μm
3. **成本倍率** — 以 IT12 为基准，如 IT7 = 4.0x
4. **相邻等级对比** — "比IT8贵170%，比IT6便宜43%"
5. **装配后果提示**
   - 间隙过大 → 晃动、噪音、精度降低
   - 过盈过大 → 装配困难、零件开裂
   - 过盈过小 → 传递扭矩不足、松动
   - 精度过高 → 成本翻倍、废品率飙升

### UI设计
在查询/推荐结果下方显示一个"工程提示"卡片：

```
┌─────────────────────────────────┐
│ 工程提示                          │
│                                   │
│ IT7 级公差                        │
│ ├ 加工方法：精车、精铣、粗磨、铰孔   │
│ ├ 表面粗糙度：Ra 0.8~1.6μm       │
│ ├ 成本：基准的 4.0 倍             │
│ ├ 对比：比IT8贵170%，比IT6便宜43% │
│                                   │
│ 💡 IT7是性价比最高的公差等级       │
│    IT7比IT8精度高一倍，成本只增50%  │
└─────────────────────────────────┘
```

## 实现步骤

### Step 1: 创建 cost-warning.js
- `getManufacturingInfo(grade)` — 返回加工方法、Ra值
- `getCostInfo(grade)` — 返回成本倍率和对比数据
- `getAssemblyWarning(fitType, grade)` — 返回装配后果提示

### Step 2: 集成到查询和推荐页面
- 在 FLOW-02 查询结果下方插入提示卡片
- 在 FLOW-03 推荐结果中附加成本提示

### Step 3: 添加独立入口
- 提供一个简单的 IT 等级选择器
- 选择后直接显示完整提示信息

## 关键数据

### 成本倍率（mechtool.cn 行业统计）
| IT12 | IT11 | IT10 | IT9 | IT8 | IT7 | IT6 | IT5 |
|------|------|------|-----|-----|-----|-----|-----|
| 1.0x | 1.2x | 1.5x | 2.0x | 2.7x | 4.0x | 7.0x | 15.0x |

### 行业经验（多方交叉验证）
- IT7 是性价比最高的等级
- 每提高一级，成本增加 20%-50%
- IT7→IT6：成本翻倍
- IT6→IT5：成本再翻倍，报废率大幅提高
